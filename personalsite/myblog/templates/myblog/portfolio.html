Blog Site

Description / Overview:

A Blog site app on Django with SummerNote as its editor. Demonstrated Django’s model and view relationship and allows for dynamic pages with pagination.

Goals and Implementation:

This project challenged my skill in working with web frameworks, the goal for this project was to have an elegant looking website that allowed me to display my blog posts to readers. Being comfortable with python i decided to use django as my framework to allow me to add onto the site in the future, django is also a great choice due to its provided paginator class that allowed for webpage layout, and its ability to integrate a rich text editor, in this case summernote, directly into its models.

Future plans/improvements:

- saving posts
- sharing on social media
- comments
- user accounts
